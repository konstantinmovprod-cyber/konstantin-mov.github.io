# PRAVDA — Telegram Mini App

Юридический сервис проверки договоров. Telegram Mini App на **React + Vite + Tailwind + @twa-dev/sdk**.

> Данные проверки пока моковые (заглушки). Бэкенд подключается позже — точки интеграции помечены в коде комментариями.

## Дизайн-система

| Токен | Значение | Назначение |
|-------|----------|-----------|
| `ink` | `#0D0D0D` | Фон (чёрный) |
| `lime` | `#C4F82A` | Акцент (кислотный лайм) |
| `card` | `#1A1A1A` | Карточки |
| `muted` | `#7A7A7A` | Второстепенный текст |
| `danger` | `#FF3B3B` | 🟥 Опасно |
| `warn` | `#F8D82A` | 🟨 Сомнительно |
| `safe` | `#C4F82A` | 🟩 Безопасно |

Шрифт — **Inter** (жирный гротеск), заголовки капсом, высокий контраст, тёмный неон, много воздуха.

## Экраны

1. **Онбординг** — лаймовая монограмма, «ЭТО PRAVDA», «Твой юрист в кармане», кнопка-пилюля «Начать».
2. **Главная** — кнопка «Проверить договор» (загрузка PDF) + список недавних документов с цветными бейджами статуса.
3. **Результат проверки** — сводка по рискам и список пунктов договора, каждый с меткой 🟥🟨🟩 и коротким пояснением.
4. **Premium** — два тарифа (350₽/мес, 2499₽/год −20%) и кнопка «Написать ГАЙД → скидка 50%».

Роутинг — простой стейт-роутер в [`src/App.jsx`](src/App.jsx) (без внешних зависимостей).

## Интеграция Telegram

Всё в [`src/hooks/useTelegram.js`](src/hooks/useTelegram.js):

- `WebApp.ready()` + `WebApp.expand()` на старте;
- header/background красятся в `#0D0D0D` под дизайн-систему;
- имя пользователя подтягивается из `initDataUnsafe.user`;
- нативные `MainButton` и `BackButton` через хуки `useMainButton` / `useBackButton`;
- хаптик-фидбек.

SDK корректно ничего не делает вне Telegram, поэтому приложение спокойно запускается в обычном браузере при разработке.

## Запуск

```bash
npm install
npm run dev       # дев-сервер
npm run build     # прод-сборка в dist/
npm run preview   # предпросмотр сборки
```

`base` в `vite.config.js` относительный (`./`), поэтому сборка работает и с корня домена, и из под-пути (например, GitHub Pages `/pravda-mini-app/`).
